## Installation

Install admin-ui using following commands:-

```bash
  npx create-react-app admin-ui
  cd admin-ui
  npm install
  npm install --save @types/styled-components
  npm start
```
    
## Deployment

To deploy this project run

```bash
  npm run build
```
